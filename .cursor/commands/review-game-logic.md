게임 로직, 특히 자동 낙하와 충돌 판정을 집중 검토해줘.

점검할 경계값/조건:

- 좌/우/바닥 벽과의 충돌 판정이 정확한가
- 이미 쌓인 블록과의 충돌 판정이 정확한가
- 조각이 보드 범위를 벗어나는 경우(인덱스 초과/음수) 처리
- 회전 시 벽이나 다른 블록과 겹치는 경우(wall kick 또는 회전 취소) 처리
- 자동 낙하 타이머가 일정하게 동작하고, 바닥 도달 시 고정(lock)되는가
- 새 조각 스폰 위치가 올바른가

각 경계 조건에 대해 현재 코드가 어떻게 처리하는지 설명하고, 누락되거나 잘못된 부분은 [버그 후보]로 표시한 뒤 재현 시나리오와 수정안을 제시해줘.
