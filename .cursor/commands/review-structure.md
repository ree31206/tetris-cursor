현재 프로젝트의 전체 구조를 검토해줘.

다음 관점에서 점검하고 결과를 표로 정리해줘:

- 파일 구성이 명확한가 (index.html, style.css, script.js의 역할 분리)
- HTML/CSS/JS 간 관심사가 잘 분리되어 있는가 (인라인 스타일/스크립트 남용 여부)
- 폴더·파일 이름이 일관적이고 직관적인가
- 불필요하거나 중복된 파일이 있는가
- 정적 사이트로 바로 실행 가능한 구조인가 (빌드 도구 없이 index.html 실행)

각 항목에 대해 [통과 / 주의 / 실패] 판정과 한 줄 근거를 제시하고, 개선이 필요하면 구체적인 수정 제안을 적어줘.
