- index.html을 브라우저로 열면 빌드 없이 바로 실행되는가
- 보드/점수/조작법 안내가 정상 표시되는가
- 핵심 기능(이동·회전·드롭·라인삭제·점수·게임오버)이 모두 동작하는가
- 콘솔 에러가 없는가
- README.md에 실행 방법, 조작법, 기능 목록, 배포 링크가 있는가
- .gitignore가 있고 불필요한 파일을 제외하는가
- 상대 경로로 리소스를 참조해 GitHub Pages에서 깨지지 않는가
